# tic-tac-toe
A cute and stylish twist on the classic Tic Tac Toe game ✨.
Designed with pastel colors, playful UI elements, and a girly aesthetic to make the experience both fun and visually delightful 💖.

🌸 Features
- 🎨 Girly Aesthetic UI: Soft pastel palette, elegant fonts, and cute icons.
- 🕹️ Classic Gameplay: Play Tic Tac Toe with the same rules you know and love.
- 👯 Two Modes:
- Player vs Player (local multiplayer)
- Player vs Computer (simple AI opponent)
- 💎 Responsive Design: Works on desktop and mobile screens.
- 🌟 Winning Animations: Sparkly effects when someone wins!

💻 Tech Stack
- Frontend: HTML, CSS (with pastel theme), JavaScript
- Optional Enhancements: React or Vue for smoother UI
- Icons/Fonts: Google Fonts (e.g., Poppins, Sacramento) + cute emoji icons
